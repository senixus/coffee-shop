# Coffee Shop

## Technologies

- Hooks
- Redux
- Redux thunk
- React router
- Firebase (firestore, auth)
- Code Splitting
- Docker
- PWA
- Unit test (Jest)
- CI/CD (CircleCI)


## Getting Started

Clone this project

`npm install` to install all required dependencies

## Firebase

Set your firebase configuration in .env file.

## Start

`npm start`
